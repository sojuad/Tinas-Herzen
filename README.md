# Weltkarte – Orte mit Link & Foto (v3)

- Google-Drive-Share-Links als Foto-URL möglich (wird automatisch in eine direkte Vorschau-URL umgewandelt).
- Hover über Marker zeigt eine Bildvorschau (Tooltip).

## Google Drive
Die Bilddatei muss auf „Jeder mit dem Link“ stehen, sonst lädt die Vorschau nicht.

## Start (lokal)
```bash
python -m http.server 8000
```
Dann: http://localhost:8000
